# Evidencia de Implementacion ICA - Modulo 3

Esta carpeta centraliza evidencia real de aplicacion en campo para los procedimientos 3.x.

## Uso rapido

1. Crear una subcarpeta por lote piloto con formato sugerido:
   - `YYYYMMDD_LOTE_PROCESO/`
2. Copiar y diligenciar:
   - `Plantilla_Acta_Piloto_Lote.md`
   - `Checklist_Evidencia_Por_Proceso.md`
3. Adjuntar soportes:
   - Foto de formato diligenciado
   - Registro CSV exportado o firmado
   - Evidencia de aprobacion de Calidad

## Minimo esperado por lote

- Identificacion de lote, proceso, fecha y responsables.
- Formato y registro asociados al POE ejecutado.
- Resultado de cumplimiento (conforme/no conforme) y accion tomada.
