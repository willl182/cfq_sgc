## Sistema de Etiquetas de Estatus

Cada lote en planta **debe** portar una etiqueta de estatus visible que indique inequívocamente su condición. **Se prohíbe** almacenar o despachar producto sin etiqueta de estatus.

## Especificaciones por Color

### ETIQUETA AMARILLA - CUARENTENA

| CAMPO | ESPECIFICACIÓN |
|:---|:---|
| **Color** | Amarillo (Pantone 116 C o equivalente) |
| **Texto principal** | "CUARENTENA - NO DESPACHAR" |
| **Tamaño mínimo** | 10 cm × 15 cm |
| **Información obligatoria** | Producto, Lote, Fecha de producción, Fecha de ingreso a cuarentena |
| **Uso** | Lote recién producido, pendiente de análisis y decisión de liberación |
| **Ubicación** | Adherida al pallet o contenedor, visible desde el pasillo |

### ETIQUETA VERDE - APROBADO

| CAMPO | ESPECIFICACIÓN |
|:---|:---|
| **Color** | Verde (Pantone 355 C o equivalente) |
| **Texto principal** | "APROBADO - APTO PARA DESPACHO" |
| **Tamaño mínimo** | 10 cm × 15 cm |
| **Información obligatoria** | Producto, Lote, Fecha de liberación, Firma/VoBo de Calidad |
| **Uso** | Lote que cumple 100% de especificaciones y documentos completos |
| **Ubicación** | Reemplaza la etiqueta amarilla en el pallet |

### ETIQUETA ROJA - RECHAZADO

| CAMPO | ESPECIFICACIÓN |
|:---|:---|
| **Color** | Rojo (Pantone 485 C o equivalente) |
| **Texto principal** | "RECHAZADO - NO USAR / NO DESPACHAR" |
| **Tamaño mínimo** | 10 cm × 15 cm |
| **Información obligatoria** | Producto, Lote, Motivo del rechazo, Fecha, No Conformidad asociada |
| **Uso** | Lote que incumple especificaciones críticas o legales |
| **Ubicación** | Adherida al pallet. Zona de rechazos segregada. |

### ETIQUETA NARANJA - APROBADO CON OBSERVACIONES

| CAMPO | ESPECIFICACIÓN |
|:---|:---|
| **Color** | Naranja (Pantone 151 C o equivalente) |
| **Texto principal** | "LIBERACIÓN CONDICIONAL" |
| **Tamaño mínimo** | 10 cm × 15 cm |
| **Información obligatoria** | Producto, Lote, Condiciones/restricciones, Responsable, Vigencia |
| **Uso** | Lote con desviaciones menores aceptadas por Dirección Técnica |
| **Ubicación** | Adherida al pallet. Requiere seguimiento. |

## Gestión de Etiquetas

| REGLA | DESCRIPCIÓN |
|:---|:---|
| **Custodia** | Calidad es el único custodio de las etiquetas de estatus |
| **Secuencia** | Todo lote inicia con AMARILLA → cambia a VERDE, NARANJA o ROJA |
| **Retiro** | Solo Calidad puede retirar o cambiar una etiqueta de estatus |
| **Registro** | Todo cambio de estatus se registra en el Formato de Liberación |
| **Destrucción** | Las etiquetas retiradas se destruyen (no se reutilizan) |
